# Roles

Student/parent:

- Browse campus, course, organization strength and teacher/research pages.
- Submit bookings and view personal booking status.
- View MakerSeed membership benefits and profile actions.

Teacher:

- View today's schedule.
- See assigned students.
- Check in students and create feedback.

Admin:

- View dashboard metrics.
- Manage bookings, students, courses, teachers, orders and memberships.

Role selection is implemented as a development-friendly entry page. Production should bind roles to WeChat login and backend authorization.
