## 配色纪律「B 表达版」（务必遵守）

| 角色 | 色值 | 用途 |
|---|---|---|
| 品牌 / 身份 | `#0079c8` 蓝 | Logo、链接、导航、选中态、信任信号 |
| **唯一行动色** | `#ff5a36` 橙 | **只**给「立即报名 / 预约体验」最高优先级按钮 |
| 结构 / 文字 | `#16121f` 墨黑 | 正文、深色区块 |
| 背景 | `#ffffff` / `#fff7f2` 暖白 + 灰 | 页面底、卡片、留白 |
| 功能色·分类 | `#6144d8` 紫 | 仅课程分类标签 / 图形母题 |
| 功能色·奖励 | `#ffb020` 金黄 | 仅会员等级 / 创客值 / 成长进度 |

原则：60% 中性 / 30% 蓝 / 10% 橙；紫与金黄不计入，只在「分类」「奖励」场景出现。

## 小程序产品结构（重要）

小程序最终分**三端**，本稿仅为**学生端**：
1. **管理端** —— 机构运营 / 排课 / 学员管理 / 财务等（待设计）
2. **教师端** —— 课表 / 点名 / 学员成长记录 / 作品评估等（待设计）
3. **学生端** —— 家长 + 学生：报名 / 课程 / 会员 / 成长（= 本稿）

学生端底部 5 Tab：**校区 · 报名 · MakerSeed · 课程 · 我的**。会员体系名「**种子科创**」，积分单位「**创客值**」。

### 2026-06-18 产品校准

用户以手机截图和后续确认覆盖了早期 Screen ③ 的会员中心方向：**MakerSeed Tab 以机构介绍 / 师资研发为当前实现准则**，并使用公司 Logo 作为底部 Tab 图标。会员体系、创客值、会员商城等保留为业务数据模型和后续功能，不作为当前 MakerSeed Tab 首屏。

当前学生端实现准则：

- `校区`：校区首页 / 本季课程故事流。
- `报名`：课程报名入口 / 学段分级 / 固定预约 CTA。
- `MakerSeed`：机构介绍，内含 `实力 / 师资研发` 切换，底部 Tab 必须保持 MakerSeed 选中。
- `课程`：课程体系，展示 5 大方向与 PBL 简章。
- `我的`：成长等级、创客值、个人功能入口。
