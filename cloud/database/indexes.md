# Database Indexes

Recommended CloudBase indexes:

- `users.openid`
- `users.role`
- `students.userId`
- `students.parentPhone`
- `teachers.userId`
- `courses.status, courses.category`
- `bookings.status, bookings.createdAt`
- `bookings.contactPhone`
- `orders.userId, orders.status`
- `memberships.userId`
- `lessonPackages.studentId`
