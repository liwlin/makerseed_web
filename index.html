🏛 江门市首家科技类培训机构 🌱 首批江门市科普教育基地

江门种子科创培训有限公司成立于 2023 年，办公场地位于万达五邑老街，配备专业教师团队。主营面向青少年的科技类教育培训与竞赛指导，并协助学校组建科创教学体系。我们已取得国家教育部要求的民办学校办学许可证与工商营业执照，依法依规开展经营。

我们推崇「造物立志」：在造物的过程中学习科学、工程、数学与设计，同时训练创造力、计算思维与解决真实问题的能力。

Learning Loop

我们怎样把兴趣变成能力

DESIGN → BUILD → CODE → DEBUG → REFLECT → IMPROVE

01

设计

从问题、限制和目标出发，画草图、拆结构、定义可验证的作品标准。

02

建造

用激光切割、3D 打印、CNC 和开源硬件，把想法变成真实的原型。

03

编程

让作品具备感知、控制、交互和联网能力，建立清晰的计算思维。

04

调试与改进

通过测试、记录、复盘和迭代，学会客观面对失败并持续优化。
